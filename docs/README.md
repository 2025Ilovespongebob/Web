## 빠른 참조

### 지도 경로 표시

#### 입력
```typescript
// 사용자가 입력하는 형식
목적지 1: lat="35.8560", lng="129.2249"
목적지 2: lat="35.8345", lng="129.2248"
목적지 3: lat="35.8290", lng="129.2180"
```

#### 출력
```typescript
// 지도에 표시되는 결과
- 마커 4개 (내 위치 + 목적지 3개)
- 경로선 3개 (빨간색)
- 총 거리: 6.5km
- 총 시간: 14분
```

### 영상 스트리밍

#### 전송 데이터
```typescript
FormData {
  frame: File (JPEG, 50-150KB),
  timestamp: "1234567890",
  frameNumber: "123"
}
```

#### 전송 주기
```
5 FPS = 200ms 간격
10 FPS = 100ms 간격
2 FPS = 500ms 간격
```

## 데이터 흐름 요약

### 지도 경로
```
사용자 입력 → 위치 배열 → 카카오 API → 경로 추출 → WebView → 지도 표시
```

### 영상 스트리밍
```
카메라 촬영 → FormData 생성 → HTTP POST → 백엔드 처리 → 응답 수신
```

## 좌표계 주의사항

### 위도 (Latitude)
- 범위: -90 ~ 90
- 방향: 남북
- 기호: Y축

### 경도 (Longitude)
- 범위: -180 ~ 180
- 방향: 동서
- 기호: X축

### 사용 예시
```typescript
// 저장 시
{ lat: 35.8560, lng: 129.2249 }

// 카카오 API
{ x: "129.2249", y: "35.8560" }

// 카카오맵 객체
new kakao.maps.LatLng(35.8560, 129.2249)
```
